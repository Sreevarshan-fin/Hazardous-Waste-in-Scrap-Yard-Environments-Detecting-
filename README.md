# Hazardous-Waste-in-Scrap-Yard-Environments-Detecting-





### 🔎 Problem Statement


Scrap yards process **1000+ tons of metal waste daily**, where hazardous items like pressurized cylinders and oil-filled shock absorbers are often hidden.
Undetected materials can cause **explosions, environmental contamination, and severe worker injuries** during processing.
Manual inspection is unreliable due to **high conveyor speeds, fatigue, and limited visibility**, leading to **15–25% miss rates**.
This creates a critical need for an **automated, real-time detection system** to improve safety and operational efficiency.
